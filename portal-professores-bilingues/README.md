# Portal dos Professores Bilíngues — MVP estrutural

Site estático inicial para o Portal dos Professores Bilíngues da GSP.

## Objetivo

Criar uma experiência navegável, gratuita e acessível para concentrar a rubrica pedagógica, materiais por segmento, planejamento, avaliação, evidências e relatórios individualizados do Projeto Bilíngue 2027.

## Princípios aplicados

- HTML navegável primeiro; PDF/Drive como apoio.
- Identidade visual própria, distinta do Portal do Projeto Bilíngue.
- Organização por segmento, disciplina e ciclo pedagógico.
- Estrutura preparada para avaliação padronizada e relatórios individualizados.

## Estrutura

- `index.html` — página principal.
- `assets/styles.css` — identidade visual estilo British bilingual academic.
- `assets/app.js` — renderização dos dados.
- `data.json` — conteúdo editável do MVP.

## Status

MVP estrutural criado para revisão de arquitetura e identidade visual.
